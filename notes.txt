# Official Documentation: https://docs.djangoproject.com/en/6.0/

# Create virtual environment to use package versions relevant to the project instead of installing these system-wide
To create: $ python3 -m venv my_env
To run: $ source my_env/bin/activate
To deactivate: $ deactivate

# Install Django
Documentation: https://docs.djangoproject.com/en/6.0/topics/install/#installing-official-release
$ python3 -m pip install Django

# Check that installation has been successful
$ python
>>> import django
>>> django.get_version()
'6.0.6'
NOTE: if the output of the django version is not the latest official version:
-- check the python version: $ python3 and $ python
-- reinstall everything again, if necessary
-- to delete a directory with all files: $ rm -r directory_name

# Creating a project
$ django-admin startproject mysite

# The below commands will create the initial database tables for the initial application
$ cd mysite
$ python manage.py migrate 

# Run the development server
$ python manage.py runserver
To quit the server: $ CTRL+C
